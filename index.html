Skills learned at the Iron Yard Academy

The Iron Yard Academy was a 12-week extensive and concentrated boot camp in which I learned a series of skills. We were sent home every night to with projects and a series of applications that we had to build and submit the next day. Every morning we had certain javascript functions in which we had to solve and figure out and create functions to answer the problem that would be in real job situations. Then we would spend the day learning a certain skill or code and also having to learn how to scour the internet to find extra resources to complete the homework assigment assigned to us. Ultimately the Iron Yard Academy did not only teach us how to code but gave me also the ability to learn different skills in a short period of time and to become independent with multiple skills.
